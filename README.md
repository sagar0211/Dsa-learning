# Dsa-learning

A small collection of basic JavaScript Data Structures & Algorithms practice exercises and patterns. The repository contains short example programs demonstrating loops, nested loops, array operations, and simple pattern printing â€” useful for beginners learning JS control flow and array manipulation.

## Repository structure

- `Loops/` - Simple loop exercises (find largest/smallest, search element index, count negatives, second largest, ...)
- `LoopsInLoops/` - Examples that use nested loops (e.g., reversing inner loops)
- `Patterns/` - Simple pattern printing examples

Example files you'll find:

- `Loops/FindLargestNumber.js`
- `Loops/FindSmallestNumber.js`
- `Loops/SearchArrayElementIndex.js`
- `Loops/CountNegatives.js`
- `Loops/SecondLargestNumber.js`
- `LoopsInLoops/ReverseInnerLoop.js`
- `Patterns/Pattern.js`

## Requirements

- Node.js (v12+ recommended)

## How to run

Open a terminal, navigate to the repository root and run any example with Node. Example (zsh):

```bash
# run a single example, e.g. find largest number
node Loops/FindLargestNumber.js

# run pattern example
node Patterns/Pattern.js
```
